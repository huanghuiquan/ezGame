Parameters:

  • id String

    这组动画栅的标识名

  • src Image | String

    图片对象或者在loader中的标识名

  • options Object

    动画栅的显示方式参数

    • x Number

      动画栅在窗口上显示位置的y坐标

    • y Number

      动画栅在窗口上显示位置的x坐标

    • beginX Number

      选择图片的起始x坐标, 默认0

    • beginY Number

      选择图片的起始y坐标, 默认0

    • frameSize Array

      帧的width和height, 默认 [60,60]

    • frameTotal Number

      帧的总数, 默认, 1

    • currentIndex Number

      当前帧的索引, 默认 0

    • duration Number

      动画栅切换帧的时间间隔单位ms, 默认100ms

    • isLoop Boolean

      是否循环显示动画栅,默认false

  • callback Function

    如果isLoop为false时，显示到最后一帧时调用callback
